# Java Average Calculator

This project is a beginner-level Java program that allows users to input any number of integers and calculates the average. Input stops when the user types "STOP".

## Features
- Accepts multiple numbers from user input
- Handles invalid input gracefully
- Calculates and displays the average
- Displays all numbers entered

## Skills Demonstrated
- Java loops and conditionals
- ArrayList usage
- Input handling with Scanner
- Exception handling (try/catch)
- Basic arithmetic and type casting
